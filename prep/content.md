# Part A: Existing Technology (Organized by Type of Tool)

# Context
Graphic design is the practice of creating visual content to communicate messages. Designers use color, typography, images, and layouts to solve problems and convey ideas. Graphic design is used in logos, posters, apps, packaging, advertising, websites, and social media. This page shows emerging graphic design tools that simplify workflows, support AI features, and provide browser-based collaboration. These tools allow beginners and professionals to quickly create visuals, prototype designs, and manage projects efficiently.

---

## 1. Full-Service Design + Prototyping Tools
Tools meant for UI/UX design, layout creation, and interactive prototypes. These tools allow designers to plan digital products, create wireframes, and test interactivity before development. They are accessible to both beginners and professionals.

### Penpot
- Open-source, browser-based design platform.  
- Supports SVG, CSS, and HTML for clean design-to-code workflows.  
- Collaborative editing in real time for remote teams.  
- Provides tools for wireframes, interactive components, and prototyping.  
- Link: [https://penpot.app/](https://penpot.app/)  
- Image: ![Penpot screenshot](https://penpot.app/assets/images/logo.svg)  

### Akira
- Linux-native vector design and UI tool.  
- Focused on UI/UX workflows and vector editing.  
- Lightweight and open-source, alternative to Adobe or Figma.  
- Link: [https://akiraux.org/](https://akiraux.org/)  
- Image: ![Akira screenshot](https://akiraux.org/images/akira-logo.png)  

### Lunacy
- Free Windows vector design tool.  
- Works offline and supports Figma file formats.  
- Includes built-in icons, stock photos, and illustration assets.  
- Link: [https://icons8.com/lunacy](https://icons8.com/lunacy)  
- Image: ![Lunacy screenshot](https://img.icons8.com/color/452/lunacy.png)

## 2. Template-Driven, Beginner-Friendly Tools
Tools focused on speed, simplicity, and ready-made assets. Useful for social media, marketing, or quick branding projects.

### RelayThat
- Automatically generates consistent layouts across multiple sizes.  
- Ideal for social media, ads, and branding.  
- AI adjusts elements while maintaining brand consistency.  
- Link: [https://relaythat.com/](https://relaythat.com/)  
- Image: ![RelayThat screenshot](https://relaythat.com/wp-content/uploads/2021/07/logo.png)  

### Snappa
- Simplified graphic creation tool for social media and ads.  
- Includes pre-sized templates for Facebook, Instagram, and YouTube.  
- Free stock photos and drag-and-drop interface.  
- Link: [https://snappa.com/](https://snappa.com/)  
- Image: ![Snappa screenshot](https://snappa.com/assets/logo.png)  

### VistaCreate (formerly Crello)
- Drag-and-drop interface with templates and assets.  
- AI features like background removal and auto-resizing.  
- Great for posters, ads, and presentations.  
- Link: [https://create.vista.com/](https://create.vista.com/)  
- Image: ![VistaCreate screenshot](https://create.vista.com/images/og-image.jpg)

## 3. Creative Typography + Vector-Focused Tools
Tools centered around typography, vector artwork, and creative design features. Useful for logos, posters, and typography-heavy projects.

### Kittl
- Browser-based typography and vector tool.  
- Useful for posters, logos, branding, and creative typography.  
- Offers templates, AI tools, and precise vector editing.  
- Link: [https://www.kittl.com/](https://www.kittl.com/)  
- Image: ![Kittl screenshot](https://www.kittl.com/images/logo.png)  

### Vectr
- Free vector graphics software for scalable images.  
- Works in-browser or as a desktop app.  
- Ideal for logos, icons, and simple vector graphics.  
- Link: [https://vectr.com/](https://vectr.com/)  
- Image: ![Vectr screenshot](https://vectr.com/images/logo.png)  

### Gravit Designer
- Cross-platform vector design app for web, desktop, and mobile.  
- Supports illustration, UI design, and logo creation.  
- Flexible export options and cloud saving.  
- Link: [https://www.designer.io/](https://www.designer.io/)  
- Image: ![Gravit Designer screenshot](https://www.designer.io/images/logo.png)
